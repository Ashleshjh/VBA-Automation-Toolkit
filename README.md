# VBA Automation and Data Engineering Portfolio

This repository contains functional VBA applications, user interfaces, and automation scripts designed to optimize data entry, validate user inputs, and manipulate Excel object models safely.

---

## 🚀 Featured Projects (Independent Architecture)

> **Disclaimer:** The tools in this section were architected, designed, and coded entirely independently to solve specific data validation and UI challenges. They are not derived from tutorial curriculums.

### 1. [Student Marks Data Entry System](./AddingStudentMarks)
A dynamic data entry interface built with ActiveX controls that interacts directly with a backend database sheet. 
* **Key Mechanics:** Utilizes a `Scripting.Dictionary` for dynamic dropdown deduplication, enforces strict/lenient data validation thresholds, and features a live-refreshing ListBox UI.

### 2. [User Registration & Skill Assessment](./UserRegistrationForm)
A structured data collection UI demonstrating boolean logic and dynamic string manipulation.
* **Key Mechanics:** Enforces mandatory field validation, processes exclusive OptionButton selections, and concatenates multiple CheckBox boolean values into a single formatted output string using mathematical trimming.
